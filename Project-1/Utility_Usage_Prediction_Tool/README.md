# ⚡ Utility Usage Prediction Tool

> A simple **Machine Learning Console Application** built with **Python** to manage utility usage records and predict future usage using **Linear Regression**.

---

## ✨ Features

* 📋 Menu-driven console application
* ➕ Add new utility usage records
* ✏️ Update existing records
* 📂 Store data in a CSV file
* 🤖 Train a Linear Regression model
* 📈 Predict future utility usage
* ⚠️ Exception handling for invalid inputs

---

## 🛠️ Tech Stack

* 🐍 Python 3
* 🐼 Pandas
* 🤖 Scikit-learn
* 📄 CSV File Handling

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone <repository-url>
```

### 2️⃣ Navigate to the Project Folder

```bash
cd Utility_Usage_Prediction_Tool
```

### 3️⃣ Create a Virtual Environment

```bash
python -m venv venv
```

### 4️⃣ Activate the Virtual Environment

**Windows**

```bash
venv\Scripts\activate
```

### 5️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Application

```bash

```
