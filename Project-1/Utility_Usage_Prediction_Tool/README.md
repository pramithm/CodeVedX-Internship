\# Utility Usage Prediction Tool



A simple \*\*Machine Learning console application\*\* built with \*\*Python\*\* that allows users to manage utility usage data and predict future usage using \*\*Linear Regression\*\*.



\---



\## Features



\* Menu-driven console application

\* Add new utility usage records

\* Update existing records

\* Store data in a CSV file

\* Train a Linear Regression model

\* Predict future utility usage

\* Exception handling for invalid inputs



\---



\## Technologies Used



\* Python 3

\* Pandas

\* Scikit-learn

\* CSV File Handling

\## Installation



1\. Clone the repository.



```bash

git clone <repository-url>

```



2\. Navigate to the project folder.



```bash

cd Utility\_Usage\_Prediction\_Tool

```



3\. Create a virtual environment.



```bash

python -m venv venv

```



4\. Activate the virtual environment.



\*\*Windows\*\*



```bash

venv\\Scripts\\activate

```



5\. Install the required packages.



```bash

pip install -r requirements.txt

```



\---



\## Run the Project



```bash

python main.py

```
